Default HTTP Header


All GET and POST request in PayWalo uses the following parameters:

Name Sample Value
Accept application/json
Content-type application/json
ClientKey [YOUR CLIENT KEY]
SecretKey [YOUR SECRET KEY]
X-PAYWALO-PLATFORM WEB
X-PAYWALO-PLATFORM values could be:
  • WEB - if the request originated from a desktop
  • MOB - if the request originated from a mobile device

This is only applicable to certain services such as online WeChat, QQ Pay and Alipay. Online banking and other options are not included and the result is the same regardless of the value passed.

All HTTP request that you will send to the PayWalo server will require that you use the above HTTP header format.
